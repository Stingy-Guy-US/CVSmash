# 📝 CVSmash – The Resume Destroyer

`CVSmash` is your personal resume blacksmith.  
One input → unlimited CV variations, all AI-generated, job-matched, and **ATS-optimized**. From minimalist design to full-blown portfolio-style resumes.  

🔧 Built with LangChain, GPT + PDF/Docx generation, and HTML magic.

---

### 🎯 Features

- 🔁 One Prompt = Multi-format CVs (PDF, DOCX, HTML)
- 🎨 Auto-branding: Insert logos, colors, and themes
- 🧠 Job-tailored: Extracts keywords from job description
- 📄 Clean + Creative resume styles
- 🗂️ Bulk CV generator (coming soon)
